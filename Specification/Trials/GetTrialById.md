## Get Trial By Id [/brapi/v1/trials/{trialDbId}]

Scope: PHENOTYPING.
Status: ACCEPTED.
Implementation target date: PAG2016.

Get trial by id.

| studies                 | array of object | List of studies                                         |          |
| studies.studyDbId       | string          | Study database identifier                               |          |
